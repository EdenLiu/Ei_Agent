# Requirement Analysis Agent

A minimal TypeScript Agent Loop for learning how model calls, tool calls, tool results, and loop termination fit together.

## Setup

```bash
npm install
```

Create `.env`:

```bash
OPENAI_API_KEY=sk-your-api-key
OPENAI_MODEL=gpt-4.1-mini
```

## Run

```bash
npm run dev
```

## Check

```bash
npm run build
```
